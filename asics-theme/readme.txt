== Description ==
WordPress theme replica of ASICS Vietnam official website.

== Installation ==
1. Upload the 'asics-theme' folder to /wp-content/themes/
2. Activate the theme through Appearance > Themes in WordPress admin

== Features ==
- Responsive design
- Custom navigation menus
- Product display templates 
- ASICS brand colors and styling
- WooCommerce compatible
- Mobile-friendly layout
